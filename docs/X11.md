## Screen

1. Ubuntu setup

   ```shell
   $ sudo vim /etc/ssh/sshd_config
   > X11Forwarding no -> yes
   > # if it didn't work
   > #X11DisplayOffset 10 -> X11DisplayOffset 10 or 11
   > #X11UseLocalhost no -> X11UseLocalhost no
   > wq (save command)
   $ sudo systemctl sshd restart
   $ sudo reboot
   ```
